# Housing Price Prediction Project

## Overview

This project implements a machine learning workflow for predicting housing prices using the California Housing dataset.

## Objectives

- Perform exploratory data analysis.
- Build a preprocessing pipeline using scikit learn.
- Train and evaluate regression models.
- Perform hyperparameter tuning with cross validation.
- Persist trained models using joblib.
